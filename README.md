# 🚀 TBZR Blog

https://tbzr.ovh    A bilingual technical blog built with **Astro** and **Starlight**.

## 🌍 Languages
- **Français 🇫🇷**
- **English 🇬🇧**

## 🛠️ Key Features
- **Framework:** [Astro 4.x](https://astro.build/)
- **Theme:** [Starlight](https://starlight.astro.build/)
- **Customization:** - Full Component Override for the language selector (added flags support).
    - Custom CSS for a unique look and feel.
    - Automatic root redirection to `/fr/`.

## 📂 Project Structure
- `src/content/docs/` : Where the magic happens (Markdown/MDX files).
- `src/components/` : Custom Astro components (like the flag selector).
- `src/custom.css` : Global style overrides.

## 🚀 Commands

All commands are run from the root of the project:

| Command | Action |
| :--- | :--- |
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your production site to `./dist/` |
| `npm run preview` | Preview your build locally |
